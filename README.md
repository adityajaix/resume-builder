# Free Resume Builder

Clean, professional resume builder that works in your browser.
No account needed. Edit directly, print as PDF.

🔗 [Try it live](https://adityajaix.github.io/resume-builder)

## Features
- Click any text to edit directly
- Live preview as you type
- Print or save as PDF — completely free
- Auto-saves in your browser
